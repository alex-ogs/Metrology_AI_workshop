# Metrology AI workshop 2026
## General description
The current project is an AI-based metrology workshop for students in the engineering school, and 
the workshop is based in the raw_tf_workloads folder.

## Restrictions
- **The project must run in a docker container**: A good practice is to keep the environment isolated, so, all the project must run in a container using the Nvidia repository.
- **Avoid Node.js and npm**
- **Changes outside the container are not allowed**
